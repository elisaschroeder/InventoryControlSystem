## Technologies Under Consideration

### Database Options
- **SQL Server**: Excellent for learning T-SQL, stored procedures, and advanced features
- **PostgreSQL**: Open-source, powerful, great for advanced SQL
- **MySQL**: Popular, widely used, good for learning fundamentals

### Application Language Options
- **C#**: Native integration with SQL Server, strong typing, excellent tooling
- **Python**: Simple syntax, good database libraries (psycopg2, pyodbc)
- **Java**: Enterprise-grade, JDBC support, cross-platform

## Challenges and Considerations

- Selecting appropriate technology stack for learning goals
- Balancing complexity vs. clarity in SQL examples
- Ensuring database design supports all planned advanced SQL concepts
- Creating realistic sample data for testing

## Resources and Learning Goals

This project aims to demonstrate proficiency in:
- Database normalization (1NF, 2NF, 3NF, BCNF)
- Complex JOIN operations (INNER, LEFT, RIGHT, FULL OUTER, CROSS)
- Subqueries and derived tables
- Common Table Expressions (CTEs) and recursive queries
- Window functions and analytical queries
- Stored procedures and functions
- Triggers for data integrity
- Indexing strategies
- Query optimization
- Transaction management and ACID properties
